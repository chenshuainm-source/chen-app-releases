## 安装

1. 下载 APK。
2. 在 Android 系统中允许对应浏览器“安装未知应用”。
3. 安装并打开「陳」，粘贴 Windows 工作台显示的 HTTPS 配对链接。

> 当前 APK 为内测签名版。本仓库只存放客户端发行文件，不存放账号、会话、文案、视频或密钥。
